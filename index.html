Data on Indian Electors, Constituencies, Polling Stations

The aim of the organization is to collect comprehensive data on Indian Electors, Constituencies, and Polling Stations.

Data

  1. Indian Assembly Constituency Shape files
  2. Electoral Rolls (pdfs)
  3. Metadata on Polling Stations (including Officers, data on building, and default link to e-roll)
  4. Indian Politician Biographies
  5. Socio-Economic Caste Census (2011)
  6. Candidate Info. + Valid Votes Won by Cands. in the 2016 Bihar Panchayat Elections

Tools

  1. Converting PDF Electoral Rolls to CSV
  2. Elector Count: Estimate the total number of electors in a state by counting the number of pages in all the electoral rolls
  3. Use Google Vision API to Get Text From (Unreadable) Electoral Rolls
  4. Use GoogleTrans to Translate Unique Non-English Cell Values in a Table and Append Their Translations
  5. Use GoogleTrans to Get Word Level Translations for Each Unique Word And Append Translated Cell Values Back
